
# RAG Model - Query Guide

## Running a Query

Follow these steps query against the model:

### 1. Clone the Repository
```bash
git clone "https://github.com/ChiefDelphi-GPT/RAG_Model.git"
cd CheifDelphi-GPT
```

### 2. Navigate to JSON Parser
```bash
cd RAG_Model/JsonParser
```

### 3. Run the Query Script
```bash
python3 user_input.py
```

### 4. Enter the Required Data

When prompted, enter the following information:

**Question:**
```
I am one of the programmers on an FRC team and I have a couple questions about how to use advantages scope. My team and I have no idea how to use it. Can you help me?
```

**Topic Slug:**
```
Advantage-Scope
```

The model will process your query and return relevant results based on the provided context.
